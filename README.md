### Подготовка проекта:
```
make install
```

### Запуск тестов:
##### 1. Запуск всех тестов
```
make test
```
##### 2. Запуск конкретного теста
```
make test type=unit name=FilterVacanciesTest
make test type=acceptance name=WebCest
```

### Запуск линтера:
```
make cs
```

### Запуск сервера:
```
make serve
```
